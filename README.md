# BugBoulevard
BugBoulevard is a small multiplayer debugging game where players compete to fix buggy code snippets as fast as possible.

# purpose
This project was built to experiment with real-time features, basic multiplayer flow, and building a game-like UI using web technologies.

# How to run
# Frontend
npm install  
npm run dev  
# Backend 
npm install  
npm run start  

# Tech used
- React  
- TypeScript  
- Vite  
- Tailwind CSS  
- Node.js  
- Express  
- Socket.IO
